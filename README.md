## Tentang Project

**UX Research Toolkit** ini dikembangkan sebagai bagian dari tugas **Metodologi Penelitian (SE-02)**. Masalah utama yang sering terjadi dalam pengembangan produk adalah proses riset yang tidak terdokumentasi dengan baik atau dilakukan secara "asal tebak".

Toolkit ini hadir untuk menstandardisasi proses tersebut, mulai dari memahami siapa penggunanya, bagaimana perjalanan mereka, hingga mengevaluasi masalah usability pada produk.

### Target Pengguna
Toolkit ini dirancang khusus untuk kebutuhan:
* **Pemda / SPBE** (Sistem Pemerintahan Berbasis Elektronik)
* **Fintech**
* **E-Commerce**
* Tim Pengembang Produk Digital

---

## Isi Toolkit (Komponen)

Repositori ini menyediakan 3 artefak utama yang siap pakai:

| Komponen | Format | Deskripsi & Fungsi |
| :--- | :--- | :--- |
| **1. Persona Template** | `.docx` | Dokumen untuk mendefinisikan profil pengguna, *goals*, *pain points*, dan *behavior* secara detail. |
| **2. User Journey Map** | `.xlsx` | Spreadsheet untuk memetakan langkah pengguna dari awal hingga akhir, termasuk emosi dan peluang perbaikan. |
| **3. Usability Checklist** | `.xlsx` | Instrumen audit untuk mengecek standar navigasi, visual, dan error handling pada aplikasi. |

## Cara Menggunakan

1.  **Clone atau Download** repositori ini.
2.  Buka folder `/files` atau klik link download di [K1 UX Research Toolkit]([https://username.github.io/repo](https://ux-research-tool.vercel.app/).
3.  **Langkah 1 (Persona):** Lakukan wawancara user, lalu rangkum datanya ke dalam *Persona Template*.
4.  **Langkah 2 (Journey):** Gunakan data tersebut untuk memetakan *User Journey Map*.
5.  **Langkah 3 (Audit):** Evaluasi desain aplikasi Anda menggunakan *Usability Checklist*.
